# Roadmap

## v0.1 ✅ (current)

- Receipt schema + validation
- SVG renderer (canonical)
- HTML renderer
- PNG renderer
- Themes (`minimal`, `cute`, `thermal`)
- React component
- CLI
- Examples

## v0.2

- Theme playground
- More card templates
- Better image handling
- Browser-side PNG export (via `@resvg/resvg-wasm`)
- Preset themes for creator booths

## v0.3

- ESC/POS command output
- 58mm / 80mm thermal layouts
- Printer adapter interface
- Test print mode

## v0.4

- Hosted receipt page mode
- QR-powered digital receipt sharing
- Coupon / revisit QR blocks
- Community theme gallery

## v0.5

- Plugin system
- Custom block plugins
- Template marketplace / community registry
